# Gestion de Candidats - Application Full Stack

## Vue d'ensemble

Application complète de gestion de candidats avec :

- API REST côté serveur (Node.js/Express/TypeScript)
- Frontend React avec TypeScript
- Base de données MongoDB
- Tests unitaires, intégration et performance

## Technologies utilisées

### Backend (Serveur)
- Node.js + Express + TypeScript
- MongoDB avec Mongoose
- JWT pour l'authentification
- Joi pour la validation
- Jest + Supertest pour les tests

### Frontend (Client)
- React + TypeScript
- React Hook Form
- Axios
- Cypress
